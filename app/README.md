# App Rol 🎲

Este proyecto forma parte del módulo de **Programación Multimedia y Dispositivos Móviles** (2º DAM). 
Es una aplicación funcional desarrollada en **Kotlin** para Android diseñada para apoyar partidas de rol de mesa.

## 🚀 Funcionalidades
* **Vídeos:** Reproductor integrado para contenido visual.
* **Sonidos:** Efectos y ambiente con control de volumen y progreso (`SeekBar`).
* **Galería:** Visualización dinámica de personajes y enemigos.
* **Dado d20:** Generador de números aleatorios con animaciones y detección de críticos.

## 🛠️ Aspectos Técnicos
* **Optimización:** Uso de `finish()` para una navegación ligera y ahorro de memoria.
* **Código Limpio:** Implementación de funciones de extensión para reutilizar el botón "Volver" en todas las pantallas.
* **Diseño:** Interfaz personalizada con paleta de colores propia e iconos adaptativos.
